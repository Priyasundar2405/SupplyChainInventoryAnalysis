# SupplyChainInventoryAnalysis
# Supply Chain & Inventory Analysis using Power BI

## Project Overview
This project focuses on analyzing inventory data across multiple warehouses to reduce stock-outs and overstocking using Power BI.

## Problem Statement
The company faces frequent stock shortages and excess inventory due to manual inventory management without analytical support.

## Objectives
- Monitor stock levels across warehouses
- Identify low-stock and overstocked products
- Optimize reorder decisions
- Improve inventory turnover rate
- Identify slow-moving products

## Dataset
The dataset consists of:
- Products table (product details and reorder point)
- Warehouses table (warehouse information)
- Inventory table (stock levels and sales data)

## Tools & Technologies
- Power BI
- DAX
- Microsoft Excel
- GitHub

## Dashboard Pages
1. Inventory Overview  
2. Shortage & Overstock Analysis  
3. Sales & Movement Analysis  

## Key Insights
- Products below reorder point were identified using stock status classification.
- Warehouses with frequent stock shortages were detected using shortage count.
- Inventory turnover rate helped measure inventory efficiency.
- Slow-moving and excess stock items were identified to reduce holding costs.
